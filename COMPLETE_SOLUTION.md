## 🔍 ТИПИЧНЫЕ ОШИБКИ И РЕШЕНИЯ

### Ошибка: "Connection timeout"
**Причина:** Сервис не запущен или не готов  
**Решение:** Запустите FireWallService от администратора и дождитесь "Waiting for GUI connections..."

### Ошибка: "Access is denied"
**Причина:** Недостаточно прав администратора  
**Решение:** Запустите оба приложения от администратора (правой кнопкой → "Запустить от имени администратора")

### Ошибка: "Empty response from service"
**Причина:** Сервис не отвечает правильно  
**Решение:** Проверьте firewall.log и перезагрузите приложение

## 🚀 КАК ИСПОЛЬЗОВАТЬ

### Быстрый старт (5 минут):
```powershell
# 1. PowerShell от администратора
Start-Process powershell -Verb RunAs

# 2. Запуск сервиса
cd "D:\Документы\MVS Programs\FireWallService\FireWallService\bin\Debug\net10.0"
.\FireWallService.exe

# 3. В другом PowerShell от администратора
cd "D:\Документы\MVS Programs\FireWallService\FirewallWiewApp\bin\Debug\net10.0-windows"
.\FirewallWiewApp.exe

# 4. В приложении: кнопка "Диагностика" → "Test Connection"
# Если видите "SUCCESS" - подключение работает! ✅
```

### Если есть проблема:
1. Нажмите "Диагностика"
2. Нажмите "Auto-Test"
3. Смотрите точное сообщение об ошибке
4. Обратитесь к [TROUBLESHOOTING.md](TROUBLESHOOTING.md)
